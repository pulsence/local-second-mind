# lsm.ui.tui

Description: Textual-based terminal user interface with screens, widgets, presenters, and state management.
Folder Path: `lsm/ui/tui/`

## Modules

- [app.py](../lsm/ui/tui/app.py): Main LSMApp Textual application
- [completions.py](../lsm/ui/tui/completions.py): Autocomplete logic
- [commands/](../lsm/ui/tui/commands/): TUI command handlers

## Sub Packages

- [screens](lsm.ui.tui.screens.md): Screen modules (query, ingest, settings, help, agents, remote)
- [widgets](lsm.ui.tui.widgets.md): Custom widgets (results, input, status, settings)
- [presenters](lsm.ui.tui.presenters.md): Presenters for screens (query, agents, settings)
- [state](lsm.ui.tui.state.md): Application state management
- [styles](lsm.ui.tui.styles.md): Textual CSS styles
